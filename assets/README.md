# Assets Klasörü

Bu klasöre aşağıdaki dosyaları eklemeniz gerekiyor:

## Gerekli Dosyalar

1. **icon.png** (1024x1024 px)
   - Uygulama ikonu

2. **splash.png** (1284x2778 px)
   - Splash screen görseli

3. **adaptive-icon.png** (1024x1024 px)
   - Android adaptive icon

4. **favicon.png** (48x48 px)
   - Web favicon

## Geçici Çözüm

Şu an için Expo varsayılan asset'lerini kullanıyor. Gerçek görselleri eklemek için:

1. Yukarıdaki boyutlarda görseller hazırlayın
2. Bu klasöre kopyalayın
3. Uygulamayı yeniden başlatın

## Not

Görseller olmadan da uygulama çalışacaktır, sadece varsayılan Expo logoları görünecektir.
