# Canlı Maç Bildirim Sesleri

Bu klasöre aşağıdaki MP3 dosyalarını ekleyin.

## Gerekli Dosyalar

| Dosya Adı | Kullanım |
|-----------|----------|
| `whistle_start.mp3` | Maç başlangıç düdüğü |
| `whistle_end.mp3` | Maç bitiş düdüğü |
| `whistle_halftime.mp3` | İlk yarı bitişi |
| `goal.mp3` | Gol |
| `yellow_card.mp3` | Sarı kart |
| `red_card.mp3` | Kırmızı kart |
| `var_review.mp3` | VAR incelemesi |
| `substitution.mp3` | Oyuncu değişikliği |
| `penalty.mp3` | Penaltı |

**Minimum:** `whistle_start`, `whistle_end`, `goal`, `yellow_card`, `red_card`

## Format

- MP3 formatında
- Kısa süreli (1-3 saniye önerilir)
- Mobil uyumlu
