# Assets Directory

## Logo

Place your shield logo image file here as `logo.png`.

The logo should be:
- PNG format (with transparency)
- Recommended size: 192x192px (will be scaled to 96x96px on auth screens, 120x120px on language screen, 160x160px on splash screen)
- Aspect ratio: Square (1:1)

Logo is used in:
- SplashScreen (160x160px)
- LanguageSelectionScreen (120x120px)
- AuthScreen (96x96px)
- RegisterScreen (96x96px)
- ForgotPasswordScreen (96x96px)
