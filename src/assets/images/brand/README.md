# Brand Assets

## Logo File Required

Place your shield logo image here:
- **File name:** `fan_manager_shield.png`
- **Format:** PNG (with transparency recommended)
- **Recommended size:** 192x192px or higher resolution
- **Aspect ratio:** Square (1:1)

## File Structure

```
src/assets/images/brand/
  ├── fan_manager_shield.png  ← ADD THIS FILE
  └── README.md
```

## Usage

The logo is used in all authentication screens:
- SplashScreen
- LanguageSelectionScreen
- AuthScreen (Login)
- RegisterScreen
- ForgotPasswordScreen

## Image Properties

- **Resize mode:** `contain` (preserves aspect ratio)
- **Sizes:**
  - Auth/Register/ForgotPassword: 96x96px
  - LanguageSelection: 120x120px
  - Splash: 160x160px
